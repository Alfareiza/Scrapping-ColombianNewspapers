## How to contribute

All code follows PEP8, except for the lenght of line, which contains 120 characters. Every function /class/method/module must have docstrings.
Please read [CONTRIBUTING.md](https://gist.github.com/PurpleBooth/b24679402957c63ec426) for details on our code of conduct, and the process for submitting pull requests to us.
Fork the project and clone or project: git clone git@github.com:<your_user>/***********.git
Install pipenv: pip install pipenv
Install the dependencies of dev: pipenv install -d
Develop the feature with tests
Run the tests locally: pipenv run pytest
Submit the pull request with testing in a single commit
Submit the PR for review
After revised and corrected, the PR will be accepted and the lib will be posted to PyPi
